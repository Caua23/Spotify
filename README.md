# Spotify Clone 1.0

Este é um clone do Spotify desenvolvido utilizando Node.js, HTML, CSS e JavaScript puro. Pretendo continuar trabalhando neste projeto no futuro, avançando ainda mais e adicionando novos recursos, possivelmente incorporando outras linguagens e tecnologias.

Desenvolvido por Vargas, também conhecido como [Caua23](https://github.com/Caua23) no GitHub.

## Recursos Futuros

- Melhorias na interface do usuário
- Adição de funcionalidades de reprodução de música
- Integração com APIs de terceiros para obter informações sobre artistas, álbuns e faixas
- Implementação de funcionalidades de criação de playlists e compartilhamento de músicas
- E muito mais!
